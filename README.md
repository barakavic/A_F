# Ascent_Fin

Ascent_Fin is a crowdfunding platform prototype designed to support Kenyan startups.  
Originally inspired by blockchain-based approaches, this version focuses on a **non-blockchain architecture** for accessibility, scalability, and ease of integration with traditional systems.

---

## Features
- Modern frontend built with Flutter/Dart (from VentureLedger adaptation).
- User-friendly interface for startup fundraising campaigns.
- Modular design to allow backend flexibility (REST APIs, databases, etc.).
- MIT Licensed for academic and open-source collaboration.
